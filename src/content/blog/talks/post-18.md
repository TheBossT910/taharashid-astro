Quick update on **SavePoint**, my video game management website for collectors and businesses.

I just completed **phase 2** with the following changes/updates:
- Added user accounts (with login and account pages)
- Add games to “To Play”, “Played”, “Owned”, “Wishlist”, “Favorites” and custom collections
- Simplified valuations/retailers section (for better user experience)
- Revamped game card design to show/hide buttons and labels
- Updated blur/colors to increase readability and consistency on website elements (including the navigation bar)
- Optimized site for performance (results in faster page loads)

<img src="/blog/talks/post-18/account.png" alt="tag" style="max-height: 800px; width: auto">

Although it doesn’t seem like much, there were a **LOT of code changes** and optimizations that needed to be done to ensure that the website loaded fast, accounts worked and were secure, and just overall clean-up of the code.

<img src="/blog/talks/post-18/collection.png" alt="tag" style="max-height: 800px; width: auto">

With phase 2 done, **I have completed all the basic collector-focused features** and will now start working on the *business-focused features* in phase 3, which will include:
- Adding business accounts
- Adding a complex inventory system which includes syncing listings with external marketplaces (eBay, Shopify, etc.)
- Creating a custom game pricing algorithm

Check out **SavePoint** at savepoint.ca
