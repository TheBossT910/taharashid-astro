# Portfolio Site (taharashid.com)

My personal portfolio and blog. The site covers work experience, personal projects, blog posts, and development logs across several projects.

**Check it out at** [taharashid.com](https://taharashid.com)

## Stack

- [Astro](https://astro.build/) - static site framework
- [Vue 3](https://vuejs.org/) - interactive components
- [UnoCSS](https://unocss.dev/) - atomic CSS
- [MDX](https://mdxjs.com/) - blog content
- [PostHog](https://posthog.com/) - analytics

## Pages

| Route | Content |
|---|---|
| `/` | Home. Intro, featured projects, recent posts |
| `/experiences` | Work history and personal/university projects |
| `/blog` | All posts (dev & blog), filterable by category |
| `/resume` | Embedded PDF resume |

**Blog categories**

- **Talks** - Articles and writeups
- **Koyomi Devlogs** - iOS/MacOS anime tracker
- **SavePoint Devlogs** — Full-stack game collection platform
- **Persona 3 Dual Devlogs** — Nintendo DS demake (C++ game engine)
